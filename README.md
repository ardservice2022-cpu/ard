# ARD Service

Ship repair, yacht and boat service – Europe. Afloat Repair Division.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Scripts

- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run preview` – preview production build
- `npm run lint` – run ESLint
- `npm run test` – run tests
